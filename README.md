# The Nineteenth Chamber

A personal tech companies ticket/ERP system built with React, Node.js, Express, and PostgreSQL.

## Tech Stack

- **Frontend**: React + Vite
- **Styling**: TailwindCSS
- **Backend**: Node.js + Express
- **Database**: PostgreSQL (planned)

## Getting Started

This project is in its initial setup phase. More details coming soon.

## Development

```bash
# Install dependencies (once setup is complete)
npm install

# Start development server
npm run dev
```
